# 四国大战 (Four Player Military Chess) - 技术规格说明书

## 1. 项目概述

- **项目名称**: 四国大战
- **项目类型**: HTML5 网页游戏
- **核心功能**: 四人军棋对战游戏，支持红蓝vs绿黄两方结盟对抗
- **目标用户**: 休闲游戏玩家

## 2. 技术规格

### 2.1 棋盘系统

| 参数 | 值 |
|------|-----|
| 画布尺寸 | 720×720 px |
| 单元格间距 (U) | 34 px |
| 玩家区域 | 4个 zone，每个 5×5 格子 |
| 布局 | 十字形排列（东南西北） |

### 2.2 坐标系统

- 原点在画布中心 (360, 360)
- `pos(g, row, col)` 函数计算每个格子的像素坐标
- g=0: 南(原位), g=1: 东(逆时针90°), g=2: 北(180°), g=3: 西(顺时针90°)

### 2.3 特殊区域

| 类型 | 数量 | 说明 |
|------|------|------|
| 行营 (Camp) | 20 | 菱形安全区，不可放地雷 |
| 大本营 (HQ) | 8 | 军旗存放位 |
| 铁路 | 8条 | 外列+外行，工兵可快速移动 |
| 中心连接 | 4 | 四个 zone 中间格连接中心 |

### 2.4 棋子配置

每人 **25 子**：

| 棋子 | 数量 | 等级值 |
|------|------|--------|
| 司令 | 1 | 10 |
| 军长 | 1 | 9 |
| 师长 | 2 | 8 |
| 旅长 | 2 | 7 |
| 团长 | 2 | 6 |
| 营长 | 2 | 5 |
| 连长 | 3 | 4 |
| 排长 | 3 | 3 |
| 工兵 | 3 | 2 |
| 炸弹 | 2 | 0 |
| 地雷 | 3 | 1 |
| 军旗 | 1 | 11 |

### 2.5 吃子规则

```
炸弹 + 任何棋子 = 同归
地雷 + 工兵 = 工兵胜
地雷 + 其他 = 地雷胜（棋子死）
军旗 = 直接获胜
大小比较：等级大的赢
同级 = 同归
```

### 2.6 移动规则

- 每次移动一格
- 工兵在铁路可飞行（不限格数，直线）
- 其他棋子铁路只能走一格

### 2.7 胜负判定

- 扛旗（吃掉对方军旗）直接获胜
- 对方全员阵亡获胜

## 4. API 接口

### 4.1 核心函数

| 函数 | 描述 |
|------|------|
| `pos(g, row, col)` | 计算像素坐标 |
| `getReachable(fromId)` | 获取可达位置列表 |
| `resolve(sp, tp)` | 吃子判定 |
| `checkVictory(player, result)` | 胜负检查 |

### 4.2 游戏状态

```javascript
{
  pieces: { cellId: { player, rank } },
  turn: 0-3,           // 当前行动玩家
  selected: cellId,    // 选中的棋子
  gameOver: boolean
}
```
